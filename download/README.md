## 🚀 安裝指南

### Windows 安裝
1. 下載 `PowerAuto.ai-v0.53-Windows.zip`
2. 解壓到任意目錄
3. 確保已安裝 Python 3.8+
4. 雙擊 `PowerAuto.ai.bat` 啟動

### macOS 安裝
1. 下載 `PowerAuto.ai-v0.53-macOS.zip`
2. 解壓文件
3. 將 `PowerAuto.ai.app` 拖拽到應用程序文件夾
4. 確保已安裝 Python 3.8+
5. 雙擊啟動應用

## 🔧 系統要求

- **操作系統**: Windows 10/11 或 macOS 10.15+
